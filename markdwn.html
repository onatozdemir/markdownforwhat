Başlık

Başlıklar kelimenin önüne numara işareti (#) konarak oluşturulur. İşaret sayısı başlık seviyesini gösterir.

Markdown HTML Output
# Lorem, ipsum. <h1>Lorem, ipsum.</h1>

Lorem, ipsum.
## Lorem, ipsum. <h2>Lorem, ipsum.</h2>

Lorem, ipsum.
### Lorem, ipsum. <h3>Lorem, ipsum.</h3>

Lorem, ipsum.

Başlık için Alternatif

1. ve 2. seviye başlıklar için sırasıyla "==" ve "--" işaretleri alt satıra tek başına konabilir.

Markdown HTML Output
Lorem, ipsum.
=====		 <h1>Lorem, ipsum.</h1>

Lorem, ipsum.
Lorem, ipsum.
--------		 <h2>Lorem, ipsum.</h2>

Lorem, ipsum.


Paragraf

Paragraflar için boş satır yeterlidir.

Markdown HTML Output
Lorem,

ipsum.		 <p>Lorem,</p>
<p>ipsum.</p>		 Lorem,

ipsum.


Satır Başı (br)

Bir satırı iki veya daha fazla boşlukla bitirip diğer satıra geçmek satır başı yapmaya yeter.

Markdown HTML Output
Lorem, ipsum. Lorem,<br><xmp>ipsum.</p> Lorem,
ipsum.


Vurgu

Bir metini bold veya italik yapmak için '*' işareti kullanılır.

Markdown HTML Output
**Lorem, ipsum.** <strong>Lorem, ipsum.</strong> Lorem, ipsum.
*Lorem, ipsum.* <em>Lorem, ipsum.</em> Lorem, ipsum.
***Lorem, ipsum.*** <strong><em>Lorem, ipsum.</em></strong></em> Lorem, ipsum.
